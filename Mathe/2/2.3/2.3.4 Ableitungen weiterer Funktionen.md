1. Trigonometrische Funktion
	$f(x) = \sin{x}$
	$f'(x) = \cos{x}$	
	Bsp.
	$f(x) = 3\sin{x} - 2x^2$
	$f'(x) = 3\cos{x} - 4x$

	$f(x) = \cos x$
	$f'(x) = -\sin x$
	Bsp.
	$f(x) = x^5 - \cos x$
	$f'(x) = 5x^4 + \sin x$

2. Exponentialfunktion
	$f(x) = e^x$
	$f'(x) = e^x$
	Bsp.
	$f(x) = 3e^x + 4\cos x$
	$f'(x) = 3e^x - 4\sin x$

3. Logarithmusfunktion
	$f(x) = \ln x$
	$f'(x) = \frac1x$
	Bsp.
	$f(x) = e^x - 5\ln x$
	$f'(x) = e^x - 5\frac1x = e^x - \frac5x$
